# entertainment-spider

漫画、动漫、小说爬虫，可观看下载。樱花动漫、风之动漫、age 动漫、看动漫、ddm 动漫、mangabz、manga5、风之漫画、mangaacg、manga90、来漫画、狼雨漫画。后续会慢慢更新其他资源网站。

# 使用方法

下载依赖包。npm run dev 开启服务器，端口 9008。然后访问 http://localhost:9008/ 即可
